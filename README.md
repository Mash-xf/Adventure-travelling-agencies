# Adventure Traveling Agencies

## Project Description
Adventure Traveling Agencies is a website project developed by Felix Macharia.  
The platform helps users book and pay for travel services offered by the agency.  

Customers can use the website to:
- Book transportation to their desired destination  
- Arrange accommodation  
- Travel for different purposes such as:
  - Visiting places  
  - Attending meetings  
  - Watching and supporting their favorite sports teams  

## Project Structure
The project consists of the following files:

- `home.html` – Main landing page  
- `contacts.html` – Contact information page  
- `sign-up.html` – User registration page  
- `log-in.html` – User login page  
- `style.css` – Styling for the website  
- `README.md` – Project documentation  

## Installation
No installation is required.  
The project is accessed directly through a web browser.
